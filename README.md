# Ahoy
Ahoy is a a Question and answers discussion board that gives its users thte ability to get their questions answered by anyone in the Ahoy community.

Technologies Used
* Node
* HTML
* CSS
* Javascript
* Mongo DB
* Express
* Mongoose
* Facebook Oauth
* Google Oauth
* Heroku
